I'm Drew, a software engineer from the United States with a background spanning full-stack web and mobile development, Web3/DeFi protocols, and embedded systems. I'm currently building HODL Bonds, an innovative DeFi protocol combining algorithmic trading strategies with the transparency and self-custody of the blockchain.

These days I'm focused on EVM smart contracts, systems-level programming in Rust, and embedded firmware, building projects like a CHIP-8 emulator or a RISC-V processor using the ULX3S FPGA dev board. I'm passionate about retro consoles, low-level computing, and bridging the gap between software and hardware.

When I'm not coding, I'm running, swinging a kettlebell, or learning Greek (Γεια σου).
